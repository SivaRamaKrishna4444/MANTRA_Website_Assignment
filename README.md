# MANTRA Educational Institutions Website

This is a beginner-friendly frontend website developed for the MANTRA 2026 Summer School assignment.

## Website Type
Educational Institution Website

## Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub
- Netlify

## Pages Included
- Home
- About
- Courses
- Programs
- Gallery
- Registration
- Contact

## Courses Offered
- Web Development
- Data Science
- Artificial Intelligence & Machine Learning
- Cyber Security

## Features
- Responsive design for mobile, tablet and desktop
- SEO tags on every page
- Educational course information
- Academic program details
- Student registration form validation using JavaScript
- Program fee calculator using JavaScript
- Interactive gallery filter using JavaScript
- Mobile navigation menu
- Image alt text for accessibility

## How to Run
Open `index.html` in any web browser.


## Netlify Deployment Steps
1. Create a GitHub repository.
2. Upload all project files.
3. Commit and push the repository.
4. Login to Netlify.
5. Click "Add New Site" → "Import an Existing Project".
6. Connect your GitHub account.
7. Select the repository.
8. Deploy the website.
